Developed at KIT by Mona Abid &  Michael Zapf in 2018

Optimized Matlab library for the XCORR processing of many 1D FFTs of same size with FFTW.


- features: autoadaptive and optimal regarding blocksize and thread usage, 
transparent and ressource-safe re-using of ressouces (wisdom, threads, plans)

- builds for linux and windows

- released under the flexible lgplv2.1, if you have questions regarding othr licenses ask.
